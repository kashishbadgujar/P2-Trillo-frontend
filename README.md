# Trillo Project  
A modern and responsive **hotel booking web interface** built using HTML, CSS and SCSS. This project focuses on learning **Flexbox layouts, UI design and advanced CSS techniques**.


## 🚀 Features  
- Responsive layout and UI animations  
- Advanced Flexbox-based design  
- Hotel listing & review UI   



## 🛠️ Technologies Used  
- HTML5  
- CSS3  
- SCSS (Sass)   



## 📂 Project Structure  
```
P2-Trillo/
├── index.html
├── css/
├── sass/
├── imgs/
└── README.md
```


## ▶️ How to Run  

1. Clone the repository:
```
https://github.com/kashishbadgujar/P2-Trillo-frontend.git
```

2. Navigate to the project folder:
```
cd P2-Trillo-frontend
```

3. Install dependencies:
```
npm install
```

4. Run:
```
npm run start
```
